# Ejercicios con kotlin

En este repositorio me centraré en resolver ejercicios básicos, pero imprescindibles para desarrollar la lógica y ganar soltura con Kotlin.

<br>

##  Estructura del repositorio :books:

Los ejercicios están guardados en archivos individuales. Cada archivo contiene en su interior el enunciado del problema comentado y la solución en código.

Algunos ejercicios tendrán una pequeña parte teórica explicada

<br>

##  Temas que desarrollaré :rocket:

-   Variables

-   Tipos de datos

-   Estructuras de control

    -   Condicionales
    -   Bucles

-   Funciones

-   Estructuras de datos
    -   Listas
    -   Tuplas
    -   Diccionarios

-   Excepciones
